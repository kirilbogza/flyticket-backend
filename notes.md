Ett IT-system för en flygleverantör. Det ska gå som admin att skapa avgångar som innehåller tidstämpel för avgång och ankomst,
från/till stad, samt antal platser i flygplanet.

Tredjepartsleverantörer så som momondo eller reseguiden kan anropa din server för att få pris för en viss avgång. De ska kunna
komma med datum för avgång (ej tid) och från/till stad och sedan få fram en biljett samt detaljer om exakt vilken tid den avgår och
vilken tid den anländer.
